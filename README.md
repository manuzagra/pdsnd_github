# New York Bike Sharing

Last update: 2026/02/13

## Description
This project is mean to build an app to handle bike sharing in the city of New York.
Users will be able to share their bikes and "rent" the bikes of other users.

## Files used
This app uses python as its main language.
bikeshare.py handles the sharing of the bikes
